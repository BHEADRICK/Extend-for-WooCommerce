## Description ##

Extend Integration for WooCommerce

## Installation ##

### Manual Installation ###

1. Upload the entire `/extend-for-woocommerce` directory to the `/wp-content/plugins/` directory.
2. Activate Extend for WooCommerce through the 'Plugins' menu in WordPress.
